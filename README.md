# Transcriptor Colab

Notebook de Google Colab para transcribir audios o videos con Whisper.

## Archivos

- `Transcriptor_10_colab_limpio.ipynb`: notebook principal para ejecutar en Google Colab.
- `Guia_uso_transcriptor_colab.docx`: guia de uso para personas que quieran usar el notebook.

## Como usarlo en Google Colab

1. Descarga o abre el archivo `Transcriptor_10_colab_limpio.ipynb` en Google Colab.
2. Conectate a un entorno de ejecucion.
3. En Colab, entra a `Entorno de ejecucion` > `Cambiar tipo de entorno de ejecucion`.
4. Configura:
   - Tipo de entorno de ejecucion: `Python 3`
   - Acelerador por hardware: `GPU T4`
5. Guarda la configuracion.
6. Ejecuta el notebook con `Ejecutar todas las celdas` o ejecuta la celda principal manualmente.
7. Sigue las preguntas que aparecen en pantalla.

Con esa configuracion deberia funcionar correctamente.

## Que permite hacer

- Tomar archivos desde una carpeta de Google Drive.
- Subir uno o varios archivos desde el computador.
- Guardar transcripciones como Google Docs, TXT descargables o ambos.
- Procesar varios archivos en lote.

## Uso recomendado

Para una prueba rapida, abre el notebook en Google Colab, elige:

1. `Subir archivos desde mi computador`
2. `Archivos TXT descargables`

Ese flujo no requiere permisos de Google Drive/Docs.
